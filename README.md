# Quips Community

Welcome to the public feedback and support hub for **Quips**, a World of Warcraft addon family focused on character speech, personality-driven reactions, companion banter, and immersive gameplay.

> This repository does not contain the Quips source code.

This repository exists so players can:

- Report bugs
- Suggest new features
- Share ideas for speech lines and behaviors
- Ask questions
- Help shape the future of Quips and its plugins

## Addons Covered Here

Feedback is welcome for:

- Quips
- Quips-Companions
- Quips-Dimwit
- Future official Quips plugins and modules

## Reporting Bugs

When reporting a bug, please include:

- Addon name and version
- World of Warcraft version
- Retail, Classic, PTR, or Beta
- Steps to reproduce the issue
- Any Lua errors
- Screenshots if applicable
- Other addons that may be involved

The more information you provide, the easier it is to investigate.

## Feature Requests

Have an idea for a new speech category, personality behavior, companion interaction, UI enhancement, or quality-of-life improvement?

Open an issue and describe:

- What you would like to see added
- Why it would improve the addon
- Any ideas for how it might work

## Discussions

GitHub Discussions may be used for:

- General questions
- Speech line ideas
- Personality ideas
- Accent ideas
- Companion behavior suggestions
- UI feedback
- Compatibility reports

## Source Code

The Quips source code is maintained separately and is not published in this repository.

This repository is intended solely for community feedback, issue tracking, and public support.
